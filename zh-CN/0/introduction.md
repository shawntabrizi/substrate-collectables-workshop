# 介绍

我们先从简单的开发环境设置开始。

此过程大约需要 15-30 分钟，具体取决于你的计算机编译 Rust 代码的速度。

在本节结束时，你将使用 `substrate-node-template` 启动自定义 Substrate 节点，并使用 Polkadot-JS Apps UI 与其进行交互。
